### 技术选型

1. 桌面应用：Electron + React (先兼容 mac 和 windows)
2. 图标库ICON：@fluentui/react-icons
3. 组件库UI: Shadcn
4. 数据库DB: Sqlite

### 功能清单：

1. 支持导入 zip 包
2. 支持填入 API Key 从而在【发现】模块显示和下载 SkillPkg的包。
3. 导入或下载包后默认弹出确认安装Dialog，默认选中所有的 Agents（可选择取消）, 点击确认即可为所有 Agents 安装。
4. 基础校验（skill的合法性）
5. 支持为指定的 Agent 安装/卸载 skill
6. 支持收藏某个 skill
7. 支持查看和修改 skill 内容
8. 自持 markdown 渲染和代码高亮

### 页面布局说明

1. 采用左右布局的方式，左侧显示菜单列表，右侧显示内容
2. 菜单页包括： 发现、本机、收藏、Agents
3. Agents下面是一个列表，显示的是当前系统安装的各类Agent工具，例如Claude, Codex, Cursor等，同时会显示对应的Agent安装的skill的数量，点击对应的tab，右侧中间栏显示的就是所有的skill列表，点击对应的skill，右侧显示的是对应的skill的目录，然后点击目录可以展开目录结构，点击具体的文件右侧显示的是文件的内容
4. 发现页显示的是从指定网站获取的skill列表（可以先写假数据模拟），包括相关的标题，介绍说明等信息，然后点击可以查看详情，详情里面有一个安装按钮，点击安装可以把skill安装到本地安装的所有的Agent工具，skill统一放到一个指定的路径存放，然后在对应的Agent工具存放skill的目录下创建一个软链接指向统一存放skill的目录，确保在当前Agent工具可以使用安装的skill，例如claude code 的在 mac 下的路径是 ~/.claude, 技能都是放在 ~/.claude/skills 中的，不同的操作系统安装路径不一样，因此需要根据windows和mac不同的操作系统做对应的处理，安装之前需要先判断本地安装了哪些Agent工具，安装的时候安装到所有的工具，如Claude, Codex, Cursor，需要根据不同的工具和不同的操作系统找到对应的工具在不同操作系统下的skill存放路径，建立对应的软链接



### 优化
 1、skill检查需要调整检查规则， 以文件夹下的skill.md文件为标准，同时读取版本号和描述
 2、刷新操作需要同时统计对应的agent下面的skill数量以及通过软链接安装的数量
 3、安装和卸载功能，如果卸载某一个skill，需要检查是否是最后一个agent工具安装了当前skill,如果是最后一个，需要提示用户是否保留当前skill
